# منصة فصيح - Faseeh Platform

## نظرة عامة

**فصيح** هي منصة ويب متخصصة في التدقيق اللغوي العميق وتحسين أسلوب الكتابة للغة العربية الفصحى. تجمع المنصة بين تقنيات الذكاء الاصطناعي المتقدمة وقواعد اللغة العربية الدقيقة لتقديم تجربة تدقيق احترافية.

## الميزات الرئيسية

### 1. المدقق اللغوي العميق (Deep Grammar Checker)
- تدقيق متقدم للنحو والصرف والإملاء وعلامات الترقيم
- كشف الأخطاء المعقدة مثل التمييز والبدل والمشتقات
- شرح مفصل لكل خطأ مع القاعدة النحوية

### 2. مؤشر فصاحة النص (Faseeh Score)
- تقييم شامل لجودة النص من 0 إلى 100
- تحليل إحصائي للمفردات وتعقيد الجملة
- توصيات مخصصة لتحسين الكتابة

### 3. قاموس المصطلحات المتخصص (Terminology Dictionary)
- قاعدة بيانات شاملة للمصطلحات الأكاديمية والقانونية والتقنية
- ترجمات دقيقة من العربية إلى الإنجليزية والعكس
- سهولة البحث والتصفح

### 4. حفظ ومراجعة النصوص
- حفظ تلقائي لجميع النصوص المدققة
- إمكانية مراجعة التدقيقات السابقة
- إحصائيات شاملة عن الأداء

## البنية التقنية

### Frontend
- **React 19** - مكتبة واجهات المستخدم
- **TypeScript** - لغة برمجة آمنة
- **Tailwind CSS 4** - تصميم واجهات متجاوبة
- **Wouter** - توجيه الصفحات
- **tRPC** - اتصال آمن مع الخادم

### Backend
- **Express.js 4** - خادم الويب
- **tRPC 11** - واجهة برمجية آمنة
- **Drizzle ORM** - إدارة قاعدة البيانات

### Database
- **MySQL/TiDB** - قاعدة بيانات موثوقة
- **Drizzle Migrations** - إدارة التغييرات

### Authentication
- **Manus OAuth** - تسجيل دخول آمن

## قاعدة البيانات

### الجداول الرئيسية

#### 1. جدول المستخدمين (users)
```sql
- id: معرف فريد
- openId: معرف OAuth
- name: اسم المستخدم
- email: البريد الإلكتروني
- role: دور المستخدم (user/admin)
```

#### 2. جدول النصوص (documents)
```sql
- id: معرف فريد
- userId: معرف المستخدم
- title: عنوان النص
- originalText: النص الأصلي
- correctedText: النص المصحح
- faseehScore: درجة الفصاحة (0-100)
- errorCount: عدد الأخطاء
```

#### 3. جدول الأخطاء (grammarErrors)
```sql
- id: معرف فريد
- documentId: معرف النص
- errorType: نوع الخطأ (نحو، إملاء، ترقيم)
- position: موقع الخطأ
- originalText: النص الخاطئ
- suggestion: الاقتراح الصحيح
- explanation: شرح القاعدة
- severity: مستوى الخطورة (low, medium, high)
```

#### 4. جدول المصطلحات (terminology)
```sql
- id: معرف فريد
- arabicTerm: المصطلح العربي
- englishTerm: المصطلح الإنجليزي
- category: الفئة (قانوني، طبي، تقني)
- definition: التعريف
```

## كيفية الاستخدام

### 1. تسجيل الدخول
- اضغط على زر "تسجيل الدخول" في الصفحة الرئيسية
- استخدم حسابك في Manus للدخول

### 2. إنشاء نص جديد
- انتقل إلى لوحة التحكم
- اضغط على "نص جديد"
- أدخل العنوان والنص
- اضغط "إنشاء وتدقيق"

### 3. عرض النتائج
- اطلع على درجة الفصاحة
- اقرأ قائمة الأخطاء المفصلة
- قارن بين النص الأصلي والمصحح

### 4. البحث عن المصطلحات
- استخدم ميزة البحث للعثور على تعريفات دقيقة
- اطلع على الترجمات والتعاريف

## التثبيت والتطوير

### المتطلبات
- Node.js 22+
- pnpm أو npm
- قاعدة بيانات MySQL

### خطوات التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/Suspect885/faseeh_platform.git
cd faseeh_platform

# تثبيت المكتبات
pnpm install

# إعداد قاعدة البيانات
pnpm db:push

# تشغيل الخادم الإنمائي
pnpm dev
```

الخادم سيعمل على `http://localhost:3000`
